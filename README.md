# RakshNetra

AI-Powered Integrated Command & Surveillance Intelligence Platform

This repository is a scaffold for RakshNetra. It provides a production-oriented Django + Channels project structure, placeholder AI model slots (the developer will upload `.pt` files later), OpenCV-based MJPEG streaming endpoints, and a dark-themed dashboard UI.

See `requirements.txt` for required packages and `README_SETUP.md` for quick start.
